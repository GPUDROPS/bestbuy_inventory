# Installation

#### [Download](https://github.com/GPUDROPS/bestbuy_inventory/releases/download/1.0.0/bestbuy_extension.zip)

#### Unzip in a location in which you want to keep it. e.g. `Documents`

<!-- #### Inside the `bestbuy_extension`, you should see:

<img width="100" alt="image" src="https://user-images.githubusercontent.com/94853139/161852499-cedac173-9398-41e7-b4e3-f9b72e431f71.png"> -->

#### Open extension settings. For Chrome, go to this url: `chrome://extensions/`

#### Enable `Developer mode` and click `Load unpacked`

   <img width="400" alt="image" src="https://user-images.githubusercontent.com/94853139/161853006-33a93ed7-f257-442b-ade8-5253600093c6.png">
